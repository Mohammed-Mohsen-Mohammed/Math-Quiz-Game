# 🧮 Math Quiz Game — C++ Console Application

![C++](https://img.shields.io/badge/Language-C++-blue)
![Status](https://img.shields.io/badge/Project-Completed-brightgreen)

> 🚀 A fully interactive math quiz game built with C++ that challenges users with dynamically generated arithmetic problems across multiple difficulty levels.

---

## 📌 Overview

This project is a **console-based math quiz system** where players solve randomly generated questions.
It focuses on **core programming fundamentals** while delivering an engaging user experience.

> This project demonstrates core C++ programming concepts such as enums, structs, functions, and control flow.

---

## 🎮 Example Gameplay

```
Question [2/5]

45 * 3
----------------
Your Answer: 135
Right Answer :-)
```

---

## ✨ Features

* 🔢 Dynamic question generation based on difficulty level
* 🎚️ Multiple difficulty modes: Easy, Medium, Hard, Mix
* ➗ Supports operations: +, -, *, /, Mixed
* 🎨 Instant feedback after each answer
* 📊 Final performance summary (Pass / Fail)
* 🔁 Replay option without restarting the program

## ⚙️ How It Works

1. User selects:

   * Number of questions
   * Difficulty level
   * Operation type

2. The system:

   * Generates random math questions
   * Evaluates answers instantly

3. At the end:

   * Displays a full performance report
